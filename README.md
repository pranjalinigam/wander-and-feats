# 🌍 Wander & Feats

A modern **Travel & Food Blogging Platform** built with **React, Vite, JavaScript, HTML, and CSS**. The project offers an engaging and responsive interface where users can explore travel destinations, discover food experiences, and enjoy a clean blogging experience.

## ✨ Features

- 🏠 Modern Landing Page
- 🌍 Travel Blogs
- 🍽️ Food Blogs
- 📖 Featured Stories
- 🌙 Dark Mode Toggle
- 📱 Fully Responsive Design
- 🎨 Clean & Modern UI
- 📩 Contact Form
- ⚡ Fast Performance with Vite
- 🧭 Smooth Navigation

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/pranjalinigam/wander-and-feats.git
```

Move into the project folder:

```bash
cd wander-and-feats
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```
http://localhost:5173
```
